# User Roles & Permissions

| Role | Scope |
|---|---|
| `master_admin` | Full system — all data nationally |
| `national_admin` | All assigned states and regions |
| `regional_admin` | Assigned regions or states |
| `state_admin` | One or more assigned states |
| `local_admin` | Assigned counties, cities, ZIP codes, or teams |
| `oa_user` | Assigned applicants, territories, locations only |

## Admin-Only Features
- Team Performance dashboard
- Territory Management
- User Management
- Admin Settings
- View all applicants across territories

## OA User Features
- Dashboard with assigned applicants
- My Numbers (own metrics only)
- All outreach and route planning
- Upload applicants (assigned territory)
- Messaging and meetings
- Reports (own data)
- Monthly reports (own data)
